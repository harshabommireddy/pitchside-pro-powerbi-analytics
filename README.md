⚽ PitchSide Pro - Executive Power BI Analytics

An end-to-end Business Intelligence project developed using **Power BI, Power Query, DAX, and Data Modeling** for the Microsoft Power BI Dataviz World Championships (Round 1).

The objective of this project is to analyze revenue performance and identify the key business drivers across products, football events, customers, regions, and acquisition channels to support executive decision-making.

This project demonstrates a complete analytics workflow:

**Data Validation → Data Modeling → DAX Development → Dashboard Design → Business Analysis → Executive Insights → Strategic Recommendations**

Project Overview

This dashboard answers the business question:

> **What is driving changes in revenue performance over time, and where should the business focus next to sustain growth?**

The analysis includes:

- Executive business performance

# 📈 Key Insights

- Revenue showed consistent long-term growth with recurring seasonal peaks.
- Football event periods generated higher daily revenue than regular trading.
- Kits generated the highest revenue, while Collectibles achieved the highest profit margin.
- EMEA contributed the highest regional revenue.
- Organic Search was the highest-performing acquisition channel.
- Long-tenure customers generated the largest share of total revenue.
